# Agent Design Placeholder

This folder will hold the CrewAI implementation after the application shell is stable.

Initial identity-aware roles:
- Manager Agent: coordinates monitoring tasks and validates recommendations.
- Performance Auditor: reviews metrics and detects performance decay.
- Security Sentinel: reviews access events and suspicious activity.
- Remediation Specialist: proposes controlled operational responses.

The first version uses deterministic mock assessment logic so the app can run without an API key.
