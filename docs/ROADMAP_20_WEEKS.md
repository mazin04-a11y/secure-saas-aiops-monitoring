# 20-Week Roadmap

## Weeks 1-4: Foundation

- Define business problem, stakeholders, requirements, and V-Model evidence.
- Implement PostgreSQL schema, FastAPI routes, and initial React dashboard.
- Add deterministic AIOps assessment logic.
- Validate local Docker runtime.

## Weeks 5-8: Agentic Layer

- Add CrewAI agent classes and task definitions.
- Implement SafeQueryTool for read-focused database access.
- Add agent execution summaries to the dashboard.
- Create test evidence for agent decisions.

## Weeks 9-12: Security and Operations

- Add access log ingestion.
- Add planned RBAC design and later authentication.
- Add incident severity workflow.
- Prepare Railway deployment.

## Weeks 13-16: DevOps and Validation

- Expand Jenkins pipeline.
- Add backend tests and frontend build checks.
- Add Docker image validation.
- Document deployment evidence.

## Weeks 17-20: Portfolio Polish

- Record demo walkthrough.
- Finalise README, diagrams, and assessment mapping.
- Export submission evidence to OneDrive.
- Reuse the template for Project 2.
